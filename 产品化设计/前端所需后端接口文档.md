# 前端所需后端接口文档

> 版本：v1.0 | 日期：2026-06-02 | 对应前端 Demo `Transwin_Demo_2.0`

---

## 一、接口约定

### 1.1 基础信息

| 项目 | 说明 |
|------|------|
| Base URL | `/api/v1`（建议，当前 Demo 使用 `/api/demo`） |
| 请求格式 | `application/json;charset=UTF-8` |
| 响应格式 | `{ "code": 200, "msg": "success", "data": ... }` |
| 认证方式 | Bearer Token（Header: `Authorization: Bearer <token>`） |
| 分页参数 | `page`（页码，从1开始）、`size`（每页条数，默认20） |

### 1.2 通用响应结构

```json
{
  "code": 200,
  "msg": "success",
  "data": { }
}
```

分页列表响应：
```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "list": [ ... ],
    "total": 100,
    "page": 1,
    "size": 20
  }
}
```

```json
{
  "marketId": 1,
  "accountId": "acc_main",
  "symbolCode": "000001",
  "orderType": 2,
  "price": 45.20,
  "quantity": 1000
}
```

```json
{

  "message": "买入委托已提交，等待成交确认。",
  "dealTime": "2026-06-02 14:30:00"
```

**请求体：**
```json
{
  "marketId": 1,
  "accountId": "acc_main",
  "symbolCode": "600519",
```

**请求体：**
```json
{
  "orderId": 10005,
  "price": 46.00,
  "quantity": 800
}
```

> `price` 或 `quantity` 至少提供一个，留空表示不修改该项。

**响应 `data`：**
```json
{
  "orderId": 10005,
  "status": "modified",
  "message": "订单修改成功。",
  "modifyTime": "2026-06-02 14:35:00"
}
```
