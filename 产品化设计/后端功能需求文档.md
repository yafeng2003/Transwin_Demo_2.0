# 后端功能需求文档

---

## 一、策略层

### 1.1 功能职责

负责交易策略的**信号生成**与**策略调度**，是交易逻辑的起点。

### 1.2 功能详情

| 功能 | 说明 |
|------|------|
| 信号生成 | 根据行情数据、技术指标、策略逻辑生成交易信号 |
| 策略调度 | 按策略配置定时或事件驱动地执行策略逻辑 |

---

## 二、执行层

### 2.1 功能职责

负责执行交易指令、同步交易结果，是连接策略与市场的桥梁。

### 2.2 功能详情

| 功能分类 | 功能项 | 说明 |
|---------|--------|------|
| 交易所接口 | 下单 | 向交易所提交委托订单 |
| | 撤单 | 撤销未成交的委托 |
| | 修改订单 | 修改已提交的委托（价格/数量） |
| | 查询持仓 | 查询当前持仓信息 |
| | 查询账户资产 | 查询账户资产信息 |
| 基础交易功能 | 市价单 | 按当前市价立即成交 |
| | 条件单 | 达到指定价格后触发买入/卖出 |
| | 止损单 | 亏损达到阈值时触发平仓 |
| | 止盈单 | 盈利达到阈值时触发平仓 |
| | 改单 | 修改已提交的委托 |
| 高级执行功能 | TWAP算法单 | 时间加权平均价格算法，将大单拆分为小单分批执行 |
| 订单与持仓同步 | 成交同步 | 从交易所获取成交结果，写入系统 |
| | 持仓同步 | 从交易所获取持仓数据，更新系统持仓 |
| 执行控制 | 重试机制 | 对失败的操作进行自动重试 |
| | 超时处理 | 对超时的委托进行超时标记和处理 |
| | 异常恢复 | 系统异常后的状态恢复 |
| 日志与记录 | 日志记录 | 记录下单、撤单、成交等所有操作的详细日志 |

---

## 三、风控层

### 3.1 功能职责

监控交易执行过程中的各类风险，进行统一风险判断、记录与通知。

### 3.2 功能详情

| 功能分类 | 功能项 | 说明 |
|---------|--------|------|
| 执行状态风控 | 下单失败监控 | 检测并记录下单失败事件 |
| | 撤单失败监控 | 检测并记录撤单失败事件 |
| | API异常监控 | 检测并记录交易所API异常 |
| 账户风控 | 单日亏损超限 | 检测单日亏损是否超过设定阈值 |
| | 最大回撤超限 | 检测账户最大回撤是否超过设定阈值 |
| | 连续亏损超限 | 检测连续亏损次数是否超过设定阈值 |
| 风控中心 | 汇总执行层状态 | 汇集执行层上报的各类事件 |
| | 汇总账户状态 | 汇集账户维度的风险指标 |
| | 统一风险判断 | 综合多维度信息进行风险判定 |
| | 统一风险记录 | 集中记录所有风险事件 |
| | 统一风险通知 | 集中发出风险通知 |
| 风险通知 | 风险报警 | 风险等级达到报警阈值时发出通知 |
| | 熔断通知 | 触发熔断时发出通知 |
| | 异常状态通知 | 系统异常状态通知 |
| | 人工操作通知 | 需要人工干预时发出通知 |

---

### 4.1 功能职责

提供统一的消息通知能力，支持邮件等多种通知渠道。

### 4.2 功能详情

| 功能 | 说明 |
|------|------|
| Email通知 | 通过邮件发送通知消息 |

---

## 五、数据分析层

### 5.1 功能职责

从数据库获取交易、资产、持仓等数据，进行分析计算，生成报表。

### 5.2 功能详情

| 功能分类 | 功能项 | 说明 |
|---------|--------|------|
| 数据获取 | 账户数据 | 从数据库获取账户资产序列 |
| | 订单数据 | 从数据库获取完整交易记录 |
| | 成交数据 | 从数据库获取成交记录 |
| | 持仓数据 | 从数据库计算某一时间点的持仓快照 |
| | 收益数据 | 从成交记录计算收益数据 |
| 数据分析 | 收益分析 | 累计收益、日/周/月收益、年化收益率、Sharpe Ratio |
| | 风险分析 | 最大回撤、回撤分析、波动率分析、风险分布 |
| | 交易分析 | 胜率、盈亏比、交易频率、手续费、滑点 |
| | 策略分析 | 多策略收益对比、风险对比、贡献度、相关性分析 |
| 报表 | 日报 | 当日交易记录、成交均价、持仓结构、仓位占比、净值曲线等 |
| | 周报 | 本周交易次数、已平仓收益、收益率排名、每日资产变化等 |
| | 月报 | 持仓平均市值、行业配置、策略绩效分析等 |
| | 报表导出 | PDF / Excel / CSV 格式导出 |

---

## 六、基础服务

### 6.1 功能职责

提供数据库、日志、定时任务等基础支撑服务。

### 6.2 功能详情

| 功能 | 说明 |
|------|------|
| 数据库 | MySQL，存储操作记录、成交记录、资产快照、完整交易记录及报表数据 |
| 日志 | 系统运行日志、交易日志、异常日志 |
| 消息通知 | Email通知（通过通知系统） |
| 定时任务 | 日报/周报/月报的定时生成与推送；资产数据的定时同步 |
